# odin-calculator
Simple calculator built as part of the Odin Foundations course.

This project was intended to apply my newfound knowledge of JavaScript by building a more intelligent tool. Although it only supports the four basic mathematical operations, creating this calculator was still immensely helpful in practicing and applying my knowledge.
The user interacts with the calculator using either their mouse or the keyboard.
